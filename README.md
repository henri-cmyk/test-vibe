# Application Calculatrice

Petite app de calculatrice en **HTML/CSS/JS** (sans dépendances), avec support clavier et quelques touches mémoire.

## Utilisation

- Ouvre `index.html` dans ton navigateur.
- Clavier : chiffres, `+ - * /`, `Entrée` (=), `Backspace`, `Esc`, `%`, `F9` (±).

## Tests (optionnel)

Prérequis : Node.js 18+.

```bash
npm test
```
